# AudioTags Enricher

Система автоматизации обработки локальной музыкальной библиотеки. Сервис сканирует аудиофайлы MP3 и FLAC, извлекает встроенные метаданные, получает дополнительные сведения из внешних музыкальных сервисов, сохраняет исходные данные в S3-совместимое хранилище MinIO и подготавливает их для очистки, нормализации и загрузки в PostgreSQL.

Система предназначена для автоматического обновления метатегов аудиофайлов и последующего перемещения обработанных треков из рабочей директории в музыкальный архив.
